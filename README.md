## 🚀 Installation

### ✅ On Replit (Recommended — One Click)

```
1. Fork this project on Replit
2. Add your Facebook cookie to account.txt
3. Set secrets: GEMINI_API_KEY and OPENROUTER_API_KEY
4. Click ▶ Run — bot starts in ~10 seconds
```

### 🖥️ On Local Machine

```bash
# Clone the repo
git clone https://github.com/CYBER-SHAKIL/Goat-bot-v3.git
cd Goat-bot-v3

# Install dependencies
npm install --legacy-peer-deps

# Add your cookie
echo "your_cookie_here" > account.txt

# Start the bot
node index.js
```

### 🔑 Required Secrets

| Secret | Where to get | Required? |
|--------|-------------|-----------|
| `GEMINI_API_KEY` | [aistudio.google.com](https://aistudio.google.com) | ✅ Yes |
| `OPENROUTER_API_KEY` | [openrouter.ai](https://openrouter.ai) | ✅ Yes |
| `GROQ_API_KEY` | [console.groq.com](https://console.groq.com) | 🔶 Optional (faster AI) |

## 🚢 Deployment

### ☁️ Replit (Recommended)

Pre-configured. Just add `account.txt` and secrets, then hit **Run**.

### 🚂 Railway

```json
{
  "build": {
    "buildCommand": "npm install --legacy-peer-deps --ignore-scripts && npm rebuild sqlite3 --build-from-source || true"
  },
  "deploy": {
    "startCommand": "node index.js"
  }
}
```

### 🟣 Render

Use the included `render.yaml`. Set `NODE_VERSION=20.20.0` in environment.
